COMO CONFIGURAR O GMAIL
1. Acesse console.cloud.google.com → New Project → "Atlantyx OS"
2. APIs → Enable → Gmail API
3. Credentials → OAuth 2.0 Client ID → Desktop App
4. Baixe o JSON de credenciais
5. Use o OAuth Playground (developers.google.com/oauthplayground) para gerar os tokens
6. Escopo: https://www.googleapis.com/auth/gmail.readonly
7. Adicione no Vercel: GMAIL_ACCESS_TOKEN e GMAIL_REFRESH_TOKEN
